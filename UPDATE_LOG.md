## 使用说明

### 启动后端
```bash
cd backend
mvn spring-boot:run
```

### 启动前端
```bash
cd frontend
npm install
npm run dev
```

### 生成测试数据（可选）
```bash
# 生成所有学生的考试成绩
node generate-exam-scores.js

# 生成考勤和日志数据
node generate-more-data.js

# 添加单个学生测试
node add-student-direct.js
```
